Fiorenza 

A simple online flower shop built with Bootstrap 5.

Features

1. Responsive layout
2. Image carousel
3. Product cards with order modals

How to Use:

1. Clone the repo: git clone https://github.com/Sanjanakiran-05/Bootstrap-fb

2. Open index.html in a browser.
